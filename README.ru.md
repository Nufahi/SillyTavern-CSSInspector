# CSS Class Inspector для SillyTavern

**Read this in other languages:** [English](README.md)

Расширение-инспектор CSS для [SillyTavern](https://github.com/SillyTavern/SillyTavern), работающее прямо в интерфейсе. Наведите курсор на любой элемент UI -- и мгновенно увидите его классы, ID, размеры, вычисленные стили, CSS-переменные и блочную модель. Копирование в один клик.

Создано для разработчиков тем, авторов расширений и всех, кто хочет разобраться в устройстве интерфейса SillyTavern без открытия DevTools.

### Плавающая кнопка инспектора (FAB)
Перетаскиваемая кнопка-прицел, парящая поверх интерфейса. Перетащите куда удобно, нажмите для включения и выключения режима инспекции.

### Инспекция элементов
- **ПК:** наведите курсор на любой элемент -- появится тултип с полной информацией. Клик - закрепить тултип, ещё клик -- скопировать.
- **Телефон (режим ПК):** работает при включённом режиме "Версия для ПК" в браузере. Тап - инспекция, повторный тап -- закрепить, третий -- скопировать.

### 5 режимов копирования

| Режим | Что копируется | Пример |
|---|---|---|
| Полный селектор | тег + id + все классы | `div#chat.flexGap5.wide100p` |
| Только классы | список классов | `.flexGap5.wide100p` |
| Только ID | id элемента | `#chat` |
| DOM-путь | полный путь от корня | `div#sheld > div.flexGap5 > div` |
| CSS-блок | готовый CSS-код | `div#chat { color: ...; }` |

### Поиск по классу
Нажмите на любой класс в тултипе — и все элементы на странице с этим классом подсветятся. Всплывающее уведомление покажет общее количество — удобно, чтобы понять, насколько широко класс используется в интерфейсе.

### Панель настроек
Все параметры настраиваются из панели расширений:

- Включить / выключить инспектор
- Тема: тёмный или светлый тултип
- Режим копирования
- Переключатели: размеры, вычисленные стили, CSS-переменные, блочная модель
- Фиксация кликом - закрепить тултип перед копированием (защита от случайных копирований)
- Уведомления - показывать или скрывать подтверждения копирования
- Сброс позиции - вернуть кнопку FAB на место по умолчанию

## Поддержка мобильных

> [!IMPORTANT]
> На телефоне необходимо включить **"Версию для ПК"** (или "Режим рабочего стола") в настройках мобильного браузера. Расширение использует hover- и click-события, которые полноценно работают только в десктопном режиме.

После включения режима ПК расширение работает так же, как на компьютере -- включая перетаскивание кнопки и инспекцию элементов.

## Установка

1. Откройте SillyTavern
2. Перейдите в **Расширения** > **Установить расширение**
3. Вставьте URL:
```
https://github.com/Nufahi/SillyTavern-CSSInspector
```
4. Перезапустите SillyTavern или нажмите `Ctrl+Shift+R`

---

## Использование

1. Включите расширение: **Расширения** > **CSS Class Inspector** > галочка **Enable**
2. На экране появится кнопка-прицел
3. Нажмите кнопку для активации режима инспекции
4. Наведите курсор на любой элемент для просмотра информации
5. Нажмите для закрепления, ещё раз - для копирования селектора

### Горячие клавиши
| Клавиша | Действие |
|---|---|
| `Escape` | Открепить тултип, отменить инспекцию |
| Правый клик | Открепить и закрыть |

## Совместимость

- SillyTavern 1.12.0+
- Десктопные браузеры: Chrome, Firefox, Edge
- Мобильные браузеры: работает в режиме "Версия для ПК" (Android Chrome и др.)
- Совместимо со всеми темами SillyTavern
